# Aune

A browser-based survival game built with vanilla HTML5 Canvas.

**[Play Aune](https://hunterbastian.github.io/aune)**

## How to Play

- **WASD / Arrow Keys** — Move
- Defeat enemies to gain XP and level up
- Choose upgrades each level to grow stronger
- Survive the endless waves

## Features

- Scrolling open world with infinite tiling grass
- 4 enemy types + boss waves
- 10 upgrades (Arcane Power, Spell Haste, Mystic Orbs, and more)
- Orbiting orbs with knockback
- Minimap, XP bar, health bar, wave announcements
- Pixel art wizard sprite
- Web Audio sound effects
- Single HTML file — no dependencies, no build step
